# ATM Management System

A console-based ATM Management System developed using Core Java to simulate basic banking operations.

## Features

- Secure PIN Authentication (3 Attempts)
- Balance Inquiry
- Cash Deposit
- Cash Withdrawal
- PIN Change
- Mini Statement
- Session Summary

## Tech Stack

- Java
- Object-Oriented Programming (OOP)

## Java Concepts Used

- Classes & Objects
- Encapsulation
- Methods
- Arrays
- Loops
- Conditional Statements
- Switch Case
- Scanner

## How to Run

```bash
javac Main.java
java atm.Main
```
